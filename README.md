# 🛡️ Defense Decision Support System (DDSS)

## 📌 Project Overview

The **Defense Decision Support System (DDSS)** is an intelligent system built using **Data Science and Machine Learning** to analyze defense-related data and provide real-time decision support.

This system helps in:

* Threat analysis
* Risk prediction
* Real-time alerts
* Decision recommendations

It improves situational awareness and supports faster, data-driven decisions.

---

## 🚀 Features

* 📊 Threat Analysis using Machine Learning
* ⚠️ Real-time Alert System
* 📡 Sensor & Intelligence Data Processing
* 📈 Risk Forecast Visualization (Chart.js)
* 🤖 Automated Decision Support
* 🔄 Dynamic UI with multiple sections

## ⚙️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript, Chart.js 
* **Backend:** Python, Flask
* **Machine Learning:** Scikit-learn / Custom ML Model
* **Data Processing:** Pandas, NumPy



## 🧠 How It Works

1. User enters:

   * Threat Level
   * Sensor Activity
   * Communication Status
   * Satellite Alerts

2. Data is sent to backend (`Flask API`)

3. Machine Learning model:

   * Predicts risk level (Low / Medium / High)
   * Generates decision support

4. Results are displayed on UI
